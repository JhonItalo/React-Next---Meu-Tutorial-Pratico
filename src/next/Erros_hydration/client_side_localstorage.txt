 
 
 
- Problema
> Ao iniciar um useState com dados vindo do localStorage, em uma context API, pode ser que os componentes que usam esse contexto, sofram com problemas de hydration.

- Solução utilizada padrão
> O Usestate foi instanciado com array vazio e no useEffect foi pego o valor do localStorage 

- Solução diferente
> Importação dinâmica do componente que utilizada desse contexto com ssr desabilitado através do dinamyc import do next
code: 

import dynamic from 'next/dynamic'
 
const DynamicHeader = dynamic(() => import('../components/header'), {
  ssr: false,
})

OBS: Problema com LocalStorage e Hydration olhe essas opções, serve para o Zustand tbm
